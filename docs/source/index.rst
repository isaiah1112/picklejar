Compatibility
-------------

As of version 3.0.0, Picklejar is compatible with the `latest supported versions of Python3!`_
If you are looking to use :code:`picklejar` with Python2.7 please use version 2.1.2.

.. _latest supported versions of Python3!: https://devguide.python.org/versions/
